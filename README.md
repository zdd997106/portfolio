- Node.js (Latest recommended)

1. **Clone the repository**  
   ```sh

   ```  

2. **Install dependencies**  
   ```sh
   pnpm install  
   # or  
   yarn install  
   # or
   npm install
   ```


